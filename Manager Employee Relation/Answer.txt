=======================================
ASSIGNMENT ON EMP AND MANAGER RELATION.
=======================================

71.WAQTD SMITHS REPORTING MANAGER'S NAME
Ans:
	SELECT ENAME FROM EMP WHERE EMPNO=(SELECT MGR FROM EMP WHERE ENAME='SMITH');
------------------------------------------------------------------------------------------------------
72.WAQTD ADAMS MANAGER'S MANAGER NAME

------------------------------------------------------------------------------------------------------
73.WAQTD DNAME OF JONES MANAGER

	SELECT DNAME FROM DEPT WHERE DEPTNO=(SELECT DEPTNO FROM EMP WHERE EMPNO=(SELECT MGR FROM EMP WHERE ENAME='JONES'));
